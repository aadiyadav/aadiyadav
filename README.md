### 🌟 Featured Project Spotlight

#### **Holiday & Attendance Management System**
*An enterprise-ready HR system designed for managing holiday calendar rules, user attendance logging, and exporting dynamic metrics.*

* 🔐 **Secure & Robust Authentication**: Complete JWT-based access control with HTTP-only cookies, password hashing via `bcrypt`, and API protection using `express-rate-limit` and `helmet`.
* 📊 **Dynamic Document Exporting**: Integrated custom PDF exports (`jspdf`, `jspdf-autotable`) and spreadsheet reports (`xlsx`) to extract user performance and history directly from React.
* 📦 **Cloud & Media Management**: Multi-part upload handler with `multer` paired with `cloudinary` storage APIs.
* 🔔 **Automated Communications**: Scheduled recurring checks via `node-cron` with email notification dispatch using `@sendgrid/mail`.
* ⚡ **Advanced State Management**: Built centralized caching and frontend state using Redux Toolkit to provide fluid screen transitions and optimistic UI updates.
